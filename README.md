# ChainMaster-Pro
INF231 –Projet de groupe sous la supervision du Pr. Melatagia.  Ce projet vise à pratiquer les différentes opérations sur les listes chaînées (simples, doubles, circulaires) en langage C.  Chaque membre du groupe est responsable d’un exercice spécifique, avec pour objectif de maîtriser l’insertion, la suppression et la gestion de listes dynamiques.
